ERPNext-Shopify Connector

Provides syncing between configured Shopify Account and ERPNext Account

ERPNext Shopify Connector will sync data between Shopify account to ERPNext account.

  1. It will sync Products and Cutomers from Shopify to ERPnext and Vice Versa.
  2. It will sync Orders from Shopify to ERPNext.
    • If payment is marked on Shopify against Order then it will create Sales Invoice is ERPNext marked as paid.
    • If fulfillment has been marked against Order in Shopify then in ERPNext it will create Delivery Note in Draft state.

Install

From your site

To install this app, login to your site and click on "Installer". Search for ERPNext Shopify and click on "Install"

Using Bench

Go to your bench folder and setup the new app

$ bench get-app erpnext_shopify https://github.com/frappe/erpnext_shopify
$ bench new-site testsite
$ bench --site testsite install-app erpnext_shopify

Login to your site to configure the app.
